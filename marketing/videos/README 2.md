## 🎬 動画の構成 (各動画共通)

```
0-3s   : Hook         ─ 大きな問いかけで興味を引く

8-13s  : Solution     ─ ZERO-PAIN なら解決! + スクショ表示

19-23s : Benefit      ─ あなたが手に入れるもの (★リスト)
23-25s : CTA          ─ ガイコツ先生 + ダウンロード誘導
```

## 📱 使い方ガイド

### Instagram Reels / Stories

#### Reels (推奨)
1. Mac から AirDrop で iPhone に転送
2. Instagram → Reels → ギャラリーから選択
3. キャプション 例:


#### Stories (3 連続投稿)
動画 1 本を 3 つに分割して、Story 3 連投もアリ:
- Part 1 (0-8s): Hook + 悩み提示
- Part 2 (8-19s): 解決策 + 使い方
- Part 3 (19-25s): 効果 + CTA

→ **滞在時間が伸びるので、エンゲージメント率 UP**

### LP (ランディングページ) への埋め込み

Vercel デプロイで動画を埋め込みも可能。
ただし容量が大きい (約 800KB-1MB/本) ので、YouTube アップロードしてから埋め込みが推奨。

#### YouTube → 埋め込み手順
1. 動画を YouTube に **非公開アップロード** (URL 知ってる人のみ視聴)
2. 「埋め込みコード」をコピー
3. LP の Hero セクションに `<iframe>` で埋め込み

→ 5 週間で 5 本 × 4 プラットフォーム = **20 投稿** で月間運用回せる。

## 🎨 動画の再生成・カスタマイズ

### スクリプト
```bash
python3 scripts/generate_promo_videos_v2.py
```

### カスタマイズ箇所
`scripts/generate_promo_videos_v2.py` の `FEATURES` 配列を編集:

```python
FEATURES = [
    {
        "id": "01-posture",
        "title": "AI 姿勢分析",
        "color": (16, 185, 129),  # 色変更可
        "hook": "あなたの姿勢、\n本当に正しい?",  # ← フックを変更
        "problem": [...],   # ← 悩みを変更
        "steps": [...],     # ← ステップを変更
        "benefits": [...],  # ← 効果を変更
    },
    ...
]
```

変更したい部分だけ書き換えて再実行すれば、新しいバージョンが生成されます。

## 📞 困ったら

- 動画の文言変更 → `scripts/generate_promo_videos_v2.py` 編集
- BGM / 音声追加したい → iMovie / CapCut で追加 (動画は素材として OK)
- 長さ短縮したい → ffmpeg で動画切り出し or スクリプトの frame 数調整
- A/B テストしたい → 別バージョン作って複数投稿
